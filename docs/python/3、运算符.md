## 赋值运算符

```python
=  直接赋值
+=  加运算并赋值
-=  减运算并赋值
*=  乘运算并赋值
/=  除运算并赋值
%=  取模运算并赋值
**=  幂运算并赋值
//=  除法取整数部分并赋值
```

## 算数运算符

```python
+  加
-  减
*  乘
/  除
%  取模
**  幂运算
//  除法取整数部分
```

## 比较运算符

```python
==  等于
!=  不等于
>  大于
<  小于
>=  大于或等于
<=  小于或等于
```

## 逻辑运算符

```python
and  且运算
or  或运算
ont  否运算
```

## 身份判断
```python
is 判断两个变量是否为同一个
is not 判断两个变量是否不为同一个
```

## 成员运算符
```python
in 判断变量是否存在某个序列（数组、字典等）
not in 判断变量是否不存在某个序列（数组、字典等）
```
